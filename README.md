# Pizzaliano
Restaurant app written in React/TypeScript and Supabase on backend.

## Features
  - Menu browsing by category
  - Cart management: add, remove, update
  - Order creation and confirmation
  - Responsive design

## Technologies Used 
  - React
  - TypeScript
  - Supabase
  - Context API
  - React Router
  - Tailwind CSS
